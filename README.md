# ☁️ Cloud Data Center Architecture & Cost Optimization

Design and evaluation of a cloud-native enterprise infrastructure by comparing Amazon Web Services (AWS) and Microsoft Azure to optimize infrastructure costs, improve scalability, and reduce Total Cost of Ownership (TCO).

---

## 📌 Overview

Modern enterprises are increasingly migrating from traditional on-premise data centers to cloud-based infrastructures to improve scalability, reliability, and operational efficiency while reducing capital expenditure.

This project evaluates the migration of an enterprise data center to the cloud by comparing Amazon Web Services (AWS) and Microsoft Azure. The analysis focuses on cloud infrastructure design, compute and storage services, pricing models, Service Level Agreements (SLAs), Total Cost of Ownership (TCO), and infrastructure scalability.

The project also investigates hybrid cloud strategies, performs sensitivity analysis, and provides recommendations for selecting the most cost-effective and reliable cloud deployment model.

---

## 🎯 Project Objectives

- Design a cloud-native enterprise infrastructure
- Compare AWS and Microsoft Azure cloud services
- Analyze cloud infrastructure costs
- Evaluate compute, storage, and networking resources
- Compare Service Level Agreements (SLAs)
- Perform Total Cost of Ownership (TCO) analysis
- Assess scalability, security, and reliability
- Recommend the optimal cloud deployment strategy
